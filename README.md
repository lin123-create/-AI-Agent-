# 上位机开发 AI Agent 辅助系统

这是一个面向公司上位机开发场景的多 Agent 辅助系统 Demo，可用于审核展示。

## 功能

- 需求分析 Agent：把客户/公司需求拆分为功能模块
- 协议解析 Agent：解析帧头、字段、命令字和校验方式
- 代码生成 Agent：生成可运行的上位机 Tkinter 框架
- 日志诊断 Agent：分析校验失败、界面卡顿、串口占用、TCP 连接失败等问题
- 报告 Agent：生成项目分析报告

## 运行方式

```bash
python upper_computer_ai_agent_assistant.py
```

Python 版本建议：3.9 或以上。

本项目默认不需要第三方库。如果使用生成后的真实串口通信代码，可安装：

```bash
pip install pyserial
```

## 审核描述建议

我构建了一个面向公司工业设备上位机开发的 AI Agent 辅助系统，主要解决需求变更频繁、通信协议复杂、界面功能重复开发、日志问题定位困难和测试报告整理效率低等问题。系统通过需求分析 Agent、协议解析 Agent、代码生成 Agent、日志诊断 Agent 和报告 Agent 协作，形成从需求输入、协议梳理、代码框架生成、异常诊断到交付文档整理的闭环流程。该系统可用于设备调试、产线测试、自动化检测和售后问题定位等场景，能够减少重复开发和人工排查时间，提高上位机项目交付效率。
